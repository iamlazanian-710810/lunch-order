# 辦公室午餐訂餐系統 — 複製建立指南

> 用 Claude 全程協助建立，約半天完成，完全免費部署。

---

## 系統功能

| 頁面 | 功能 |
|------|------|
| 主頁 `/` | 員工選名字 → 看今日菜單圖片 → 手填餐點/價格/備註 → 送出 |
| 後台 `/admin` | 上傳今日菜單圖、查看即時訂單、複製叫餐清單（需 PIN 碼） |
| 後台本月記錄 | 依日期展開每天所有人的訂單明細 |
| 報表 `/report` | 選月份 → 每人費用總表 → 可展開每日明細 → 匯出 CSV |

---

## 技術架構

| 項目 | 選擇 | 費用 |
|------|------|------|
| 前後端 | Next.js 16（App Router） | 免費 |

| 身分識別 | 下拉選名字（無帳密） | — |

## 日常使用流程

```

  → 今日設定 → 複製叫餐清單 → 打電話訂餐
```
